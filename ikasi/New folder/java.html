JAVA

In this tutorial I teach you the fundamentals of Java programming, in just under 15 minutes. Learn if/else, loops and even inheritance and composition. Obviously, you can't master a language in just 15 minutes - but its a good start.















I specifically cover the following topics: primitive data types, comments, class, import, Scanner, final, Strings, static, private, protected, public, constructors, math, hasNextLine, nextLine, getters, setters, method overloading, Random, casting, toString, conversion from Strings to primitives, converting from primitives to Strings, if, else, else if, print, println, printf, logical operators, comparison operators, ternary operator, switch, for, while, break, continue, do while, polymorphism, arrays, for each, multidimensional arrays.




Learn to use Apache Hadoop to build applications to analyse Big Data. Understand => The Hadoop Distributed File System (HDFS) => Setup, manage and monitor Hadoop cluster => MapReduce - Combiners, Partitioners, MapSide Joins => Apache Sqoop => Apache Hive, run HiveQL queries to create tables => Apache Pig, Pig Latin scripting language => Apache Flume, how to load unstructured data into HDFS => Apache HBase

Sequence of three tutorials to get an inside-out knowledge of JAVA
